1. 기본 모듈 (Basic Modules)

ZW3D에서 공통적으로 사용되는 인터페이스, 구조체, 열거형 등을 포함합니다.

1.1 일반 데이터 (General Data)

공통 인터페이스 및 구조 정의
헤더 파일: zwapi_util.h

1.2 환경 설정 (Configuration)

ZW3D의 설정 정보를 가져오거나 수정
기본 설정 파일: supp/config.xml
헤더 파일: zwapi_config.h

1.3 전역 애플리케이션 설정 (Global Application)

단위, 색상, 버전, 언어, 스케치 그리드 설정 등
헤더 파일: zwapi_global_apply.h

1.4 라이선스 (Authorization)

인증 및 라이선스 관련 API
헤더 파일: zwapi_license.h

1.5 디스플레이 (Display)

뷰 영역 표시/숨김, 조명, 배율 설정 등
헤더 파일: zwapi_display.h

1.6 메시지 처리 (Message)

오류, 경고 메시지 출력 등
헤더 파일: zwapi_message.h

1.7 뷰 (View)

일반적인 뷰 관련 API
헤더 파일: zwapi_view.h

1.8 메모리 (Memory)

메모리 해제 관련 API
헤더 파일: zwapi_memory.h

1.9 파일 루트 (File Root)

파일 루트 및 단일/다중 루트 파일 처리
헤더 파일: zwapi_root.h

1.10 레이어 (Layer)

레이어 API
헤더 파일: zwapi_layer.h

1.11 이벤트 (Event)

실행 취소, 다시 실행 관련 API
헤더 파일: zwapi_xn.h

1.12 일반 치수 API (Standard General API)

치수 속성 및 텍스트 처리
헤더 파일: zwapi_dim.h

1.13 일반 수식 API (Expression General API)

수식 처리 API
헤더 파일: zwapi_var.h

1.14 일반 텍스트 API (Text General API)

텍스트 속성 및 처리
헤더 파일: zwapi_text.h

1.15 엔티티 일반 API (Entity General API)

객체 이름, ID, 색상 설정 등
헤더 파일: zwapi_general_ent.h

1.16 사용자 입력 (User Interaction Module)

사용자의 선택 및 입력 처리
헤더 파일: zwapi_userinput.h

2. 파일 모듈 (File Module)

파일 열기, 닫기, 저장, 가져오기/내보내기 등과 관련된 API

2.1 파일 데이터 구조 (File Data)

헤더 파일: zwapi_file_data.h

2.2 파일 연산 (File Operation)

파일 조작 API
헤더 파일: zwapi_file.h

2.3 경로 처리 (Path Operation)

경로 및 검색 경로 설정
헤더 파일: zwapi_file_path.h

3. 토폴로지 엔티티 모듈 (Topology Entity Module)

파트 환경에서 브렙(BRep) 엔티티 생성과 관련된 객체 처리

3.1 Brep 데이터

3.2 형상 API (Shape)

형상 속성 조회

3.3 면 API (Face)

서피스 속성 조회
헤더 파일: zwapi_brep_face.h

3.4 엣지 API (Edge)

엣지 속성 조회
헤더 파일: zwapi_brep_edge.h

3.5 UV 엣지 API

헤더 파일: zwapi_brep_preedge.h

3.6 버텍스 API (Vertex)

헤더 파일: zwapi_brep_vertex.h

4. 알고리즘 모듈 (Algorithm Module)

기하학 계산, 매트릭스 연산, 간섭 체크 등

4.1 수학 데이터 (Math Data)

헤더 파일: zwapi_math_data.h

4.2 일반 연산

헤더 파일: zwapi_math.h

4.3 바운딩 박스 계산

헤더 파일: zwapi_math_bndbox.h

4.4 간섭 체크 (Interference)

헤더 파일: zwapi_math_insect.h

4.5 점 관련 알고리즘

헤더 파일: zwapi_math_point.h

4.6 벡터 연산

헤더 파일: zwapi_math_vector.h

4.7 행렬 연산

헤더 파일: zwapi_math_matrix.h

UI 구성요소 제어용 API

6. 파트 객체 모듈 (Part Object Module)

파트 환경에서 모델 객체 조회, 설정, 속성 등

7. 어셈블리 객체 모듈 (Assembly Object Module)

조립 환경 관련 객체 및 속성 API

구성 요소 속성: zwapi_asm_comp_attribute.h
삽입/조립: zwapi_asm_comp.h
구속 조건: zwapi_asm_constraint.h
재사용 라이브러리: zwapi_asm_reuselibrary.h

8. 도면 시트 모듈 (Drawing Sheet Object Module)

2D 도면 관련 객체 및 속성 처리

도면 속성: zwapi_dwg_attribute.h
뷰, 치수, 기호, 표, 도형 등: 각종 zwapi_dwg_*.h

9. 스케치 객체 모듈 (Sketch Object Module)

스케치 환경에서의 객체 생성 및 처리

데이터 구조: zwapi_sk_data.h
지오메트리 객체: zwapi_sk_objs.h
치수: zwapi_sk_dim.h
수식: zwapi_sk_var.h
연산: zwapi_sk_opts.h

10. CAM 모듈 (CAM Module)

가공 환경 관련 속성, 기계, 공정, 도구 생성 및 제어

파트/트리/기계/공정 등: zwapi_cm_*.h

11. 명령어 생성 (Command Creation)

리본 메뉴 기준 명령어 분류 및 정의

공통 데이터 및 명령: zwapi_cmd.h, zwapi_cmd_data.h
파트/어셈블리/시트/스케치 명령: zwapi_cmd_*.h

12. 도구 모듈 (Tool Module)

엔티티 경로 등 기본 도구 API

엔티티 경로: zwapi_tool_entpath.h

13. PMI (Product Manufacturing Information)

치수 정보, 기호 등 제조 정보 처리

PMI 객체 조회: zwapi_pmi.h

14. 하니스 (Harness)

와이어링/하니스 관련 API

하니스 처리: zwapi_harness.h

ZW3D 2025 AI 기능

ZW3D 2025에서는 AI 기반 도구들이 추가되어 API 개발 효율성이 크게 향상되었습니다:

  • Command Advisor: 사용자 행동 분석을 통한 효율적인 설계 지원
  • API 어시스턴트: 자연어를 통한 API 코드 자동 생성
  • 모델 매칭: 3D 모델 자동 식별 및 검색
