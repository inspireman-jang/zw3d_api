7장: ZW3D 특수 기호 이스케이프 문자 맵

📝 중요 참고사항:

ZW3D 기호 열의 큰따옴표 안의 텍스트는 ZW3D 특수 기호에 해당합니다. '31'은 ASCII 코드를 사용하며, 입력할 때 특별한 처리가 필요합니다.

  • '31'의 16진수 ASCII 코드: 0x1F
  • '127'의 16진수 ASCII 코드: 0x7F

🔤 사용 방법

이 문자 맵은 ZW3D에서 특수 기호를 표시하거나 입력할 때 사용됩니다. 각 기호는 다음과 같은 형식으로 구성됩니다:

형식: [VxX]문자 → 유니코드 → 표시 기호
예시: [VxX]# → \U+E012 → 특수 기호

💡 사용 예제

기본 사용법:
텍스트에서 "[VxX]#"를 입력하면 → 특수 기호로 변환됨
텍스트에서 "[VxY]$"를 입력하면 → 다른 스타일의 특수 기호로 변환됨

⚠️ 특별 주의사항:

  • '31' 및 '127'은 ASCII 코드 값으로, 직접 입력 시 특별한 처리가 필요합니다
  • 일부 기호는 특정 시리즈에서만 사용 가능합니다
  • 유니코드 값은 시스템 폰트에 따라 표시가 다를 수 있습니다

🔍 시리즈별 특징

VxX 시리즈

기본적인 ZW3D 특수 기호 세트로, 가장 널리 사용되는 기호들을 포함합니다.

VxY 시리즈

확장된 기호 세트로, VxX 시리즈의 대안적 표현이나 추가 기호들을 제공합니다.

VxWD 시리즈

특수 목적의 기호 세트로, 제한적인 기호들만 포함하며 주로 방향 표시 등에 사용됩니다.

📚 추가 정보

이 문자 맵은 ZW3D 내에서 도면 작성, 주석 추가, 특수 표기 등에 사용됩니다. 각 기호의 정확한 표시는 사용하는 시스템과 폰트에 따라 달라질 수 있으므로, 실제 사용 전에 ZW3D 환경에서 테스트해보는 것을 권장합니다.
