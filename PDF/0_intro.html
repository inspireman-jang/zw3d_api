ZW3D API 소개

이 문서는 ZW3D API에 대한 기본 소개를 제공합니다.

ZW3D 기반의 애드온 개발을 위해 C/C++를 권장합니다. 파트너들이 ZW3D 기반의 강력한 애플리케이션을 개발할 수 있도록 API를 지속적으로 개선하고 있습니다.

1장: ZW3D API 학습

주요 내용:

  • ZW3D 2차 개발과 관련된 메커니즘 소개
  • 개발 시 주의사항 및 베스트 프랙티스
  • 구 인터페이스와 신 인터페이스 간의 차이점
  • API 사용 시 필수 고려사항

2장: ZW3D API 프로젝트 생성

주요 내용:

  • 간단한 프로젝트 생성 방법 단계별 가이드
  • 프로그램 디버깅 방법 및 도구 사용법
  • 개발 과정에서 주의해야 할 사항들
  • ZW3D API를 빠르게 시작하는 방법
  • 개발 환경 설정 및 구성

  • 필요한 기능 버튼을 빠르게 배치하는 방법
  • 이전 버전에서 기능 마이그레이션 지원
  • 구성 파일을 빠르게 업그레이드할 수 있는 도구 제공
